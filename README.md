# Mlaku Mulu Infra

Infra for Mlaku Mulu operations.

## Technology Stack

- MySQL
- Git
- Docker

## Installation

### 1. Environment Setup

```bash
git clone https://github.com/heryrmdn/mlaku-mulu-infra.git
cd mlaku-mulu-infra
```

### 2. Environment Variables

Create `.env` file from template:

```bash
cp .env.example .env
```

Fill in appropriate values in the `.env` file.

### Docker

```bash
docker compose up -d
```
